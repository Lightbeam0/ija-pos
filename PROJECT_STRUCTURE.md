## Notes

- `node_modules/` is intentionally excluded because it is generated by `npm install`.
- `.env` contains local environment values and should not be committed.
- `.env.example` documents the environment variables needed to run the project.
- `prisma/schema.prisma` defines the database schema.
- `prisma/migrations/` contains database migration history.
- `src/index.ts` is the application entry point.
- `src/routes/` contains API route handlers.
- `src/middleware/` contains Express middleware.
- `src/lib/prisma.ts` contains the Prisma client setup.
- `src/prisma/seed.ts` contains database seed logic.
